发表于 2017-09-19 | 分类于 coding | | 阅读次数

Babel 作为一个能够让我们愉快地用上 ES6 的工具，大大地解放了我们的“生产力”。但是，在我使用 Babel 的时候，总是很好奇，Babel 到底把我们的代码转换成什么样子了呢？会不会在其中会有什么坑呢？所以在本文中，我们会针对一些测试用例，来查看一下 Babel 转换的结果到底是什么。
